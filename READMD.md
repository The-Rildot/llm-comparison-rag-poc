# LLM Comparison RAG POC

Proof-of-concept project comparing multiple LLM providers using a shared
retrieval-augmented knowledge base.

**Goal:** Evaluate accuracy, latency, cost, and hallucination behavior across
different AI models using identical data and prompts.

## Tech Stack
- FastAPI (Python)
- ChromaDB
- sentence-transformers
- OpenAI API
- Local LLMs (Ollama)
